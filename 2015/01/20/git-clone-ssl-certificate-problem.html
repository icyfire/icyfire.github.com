Git clone时报SSL certificate problem错误的解决方法

用Git安装某应用时报了如下错误：

$ git clone https://github.com/phacility/arcanist.git
Cloning into 'arcanist'...
error: SSL certificate problem, verify that the CA cert is OK. Details:
error:14090086:SSL routines:SSL3_GET_SERVER_CERTIFICATE:certificate verify failed while accessing https://github.com/phacility/arcanist.git/info/refs?service=git-upload-pack
fatal: HTTP request failed

应该是SSL证书的验证问题，我们可以通过环境变量把Git的SSL验证关掉来解决这个问题：

$ export GIT_SSL_NO_VERIFY = true
