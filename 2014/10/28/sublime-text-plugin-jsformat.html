Sublime Text插件 —— JsFormat

概述

JsFormat是Sublime Text 2上基于http://jsbeautifier.org/开发的一个插件。可以用它来格式化整个js或者json文件，也可以对一个文件中的某一部分进行格式化。

Github: https://github.com/jdc0589/JsFormat

只要使用Ctrl + Alt + f这个快捷键，当前的js文件或者选择的代码就会立刻格式化：

安装

直接使用Package Control进行安装。Ctrl + Shift + p呼出控制面板：

配置

JsFormat使用了translate_tabs_to_spaces和tab_size这两个Sublime Text的配置项，另外还提供了以下的配置项：

{
    // exposed jsbeautifier options
    "indent_with_tabs": false,
    "preserve_newlines": true,
    "max_preserve_newlines": 4,
    "space_in_paren": false,
    "jslint_happy": false,
    "brace_style": "collapse",
    "keep_array_indentation": false,
    "keep_function_indentation": false,
    "eval_code": false,
    "unescape_strings": false,
    "break_chained_methods": false,
    "e4x": false,
    "wrap_line_length": 0,

    // jsformat options
    "format_on_save": false,
    "jsbeautifyrc_files": false
}
配置文件在“Preferences->Package Settings->JsFormat”下，你可以通过修改"Settings - User"去修改配置。
各配置项的具体含义请参看jsbeautifier官网的文档。

如果jsbeautifyrc_files这个配置项为true，则可以使用.jsbeautifyrc配置文件，.jsbeautifyrc是个JSON文件，支持所有JsBeautifier配置项。配置文件的优先级为default options -> user settings -> .jsbeautifyrc配置文件。

关于.jsbeautifyrc具体信息，请查看以下链接：https://github.com/jdc0589/JsFormat#jsbeautifyrc-files

快捷键

JsFormat默认的快捷键为Ctrl + Alt + f，如果有其他插件也用了这个快捷键的话，就会产生冲突。可以使用以下方式解决：

1. 修改“Preferences->Key Bindings - User”，添加如下语句：{ "keys": ["ctrl+alt+f"], "command": "js_format"}
2. 修改冲突的插件的Key Bindings文件，把冲突的快捷键修改为其他。
