GO —— Hello World

安装

Linux下安装Go的系统需求:

Operating system Architectures Notes
Linux 2.6.23 or later with glibc amd64, 386, arm CentOS/RHEL 5.x not supported; no binary distribution for ARM yet

使用源码安装的方式进行安装，这里安装的是1.3.1版本。

$ wget https://storage.googleapis.com/golang/go1.3.1.src.tar.gz
# 安装到/usr/local/services里
$ tar zxvf go1.3.1.src.tar.gz - C /usr/local/services/
# 进入源码目录，运行all.bash进行安装
$ cd /usr/local/services/go/src
$ ./all.bash

安装完设置环境变量，在/etc/profile里增加以下两行：

export GOROOT=/usr/local/services/go
export PATH=$PATH:$GOROOT
Go默认你的安装路径是/usr/local/go，所以如果是安装在默认路径下，只需要增加一行：export PATH=$PATH:/usr/local/go/bin，而不需要设置GOROOT这个环境变量。

Hello World

下面编写一个Hello World来验证我们的安装。创建一个名为hello.go的文件并编写以下代码：

package main

import "fmt"

func main() {
    fmt.Println("hello, world.")
}

使用go工具运行这段代码：

$ go run hello.go
hello, world.

看到hello.worlds.则表示Go安装成功。

更详尽的安装信息请参考这个链接：http://godoc.golangtc.com/doc/install
